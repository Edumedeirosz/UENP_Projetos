# UENP_Projetos

Repositório contendo projetos e trabalhos realizados durante o curso de Computação na Universidade Estadual do Norte do Paraná (UENP).

Cada pasta corresponde a um projeto ou exercício diferente, com seu próprio README detalhando:
- Objetivo do projeto ou também conteúdos aprofundando-os
- Tecnologias utilizadas
- Como executar ou testar

Alguns projetos incluem:
- Exercícios de Java e Orientação a Objetos
- Testes unitários com JUnit
- Trabalhos acadêmicos de lógica e programação

> Nota: Este repositório é de caráter acadêmico, usado para estudo e demonstração das habilidades adquiridas no curso, geralmente feitas/enviadas por interesse próprio.
